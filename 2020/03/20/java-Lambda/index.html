什么是 Lambda表达式？

它是匿名函数的简洁表示，可以传递；不需要进行定义，也不需要与特定类关联。Lambda表达式是一种简洁的表示形式，因为不需要像编写匿名类那样编写样板代码。

将方法转换为Lambda表达式:

  • 去除方法名
  • 去除返回类型定义
  • 去除修饰符
  • 添加箭头符号（->）
