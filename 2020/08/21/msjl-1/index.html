  1. char 型变量中能不能存储一个中文汉字，为什么

char 类型可以存储一个中文汉字，因为 Java 中使用的编码是 Unicode（不选择任何特定的编码，直接
使用字符在字符集中的编号，这是统一的唯一方法），一个 char 类型占 2 个字节（16 比特），所以放一个中文是没问题的。

ACID
原子性 事务是一个完整的操作
一致性 事务内的操作要么一起失败，要么一起成功
隔离性 多个事务是并发操作，不相互影响
持久性 修改能永久保存
